Projects

Stocks Project

Designed stock simulator platform that allows you to buy/ sell stocks and make portfolios. Uses the MVC model, the command design pattern. Has a GUI and a text-based UI.

MineSweeper Project

Designed Minesweeper game using Java. Tracks the number of unflagged mines and covered tiles. Board size is adjustable and the number of mines change according to board size.

Light Em All Project

Designed Light Em All Game using Java. Used Kruskal's Algorithm to randomize the placement of the pipes while still ensuring the game is winable. Calculates and displayes the number of clicks allowed depending on board size.
